ВИДЕО ДЛЯ САЙТА — КУДА ЗАГРУЖАТЬ
=================================

Положите СЮДА (в папку uploads/) четыре файла с ТОЧНО такими именами:

  video1.mp4   →  большое видео в hero-блоке (вверху, играет само, без звука, зациклено)
  video2.mp4   →  1-й ролик в разделе «Видео»
  video3.mp4   →  2-й ролик в разделе «Видео»
  video4.mp4   →  3-й ролик в разделе «Видео»

ВАЖНО:
- Имена файлов ровно такие: video1.mp4, video2.mp4, video3.mp4, video4.mp4 (маленькими буквами).
- Формат — .mp4 (кодек H.264). Так видео открывается во всех браузерах и телефонах.
- Каждый файл держите МЕНЬШЕ 25 МБ — иначе GitHub не даст загрузить через сайт.
  Если файл больше — сожмите его (см. ниже).
- После загрузки файлов обновите страницу — видео появятся сами.

КАК СЖАТЬ ВИДЕО (если больше 25 МБ):
- Онлайн: freeconvert.com/video-compressor или clideo.com/compress-video
- Или на телефоне любое приложение «сжать видео» — выберите 720p.

КАК ЗАГРУЗИТЬ НА GITHUB:
1. Откройте свой репозиторий на github.com
2. Зайдите в папку uploads (или создайте её кнопкой Add file → Create new file, введя uploads/video1.mp4 — но проще Upload files)
3. Add file → Upload files → перетащите video1.mp4 ... video4.mp4
4. Commit changes
5. Vercel пересоберёт сайт сам — видео заработают у всех.
